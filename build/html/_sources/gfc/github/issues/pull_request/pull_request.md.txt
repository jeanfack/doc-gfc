# Pull Request

::::{grid}
:::{grid-item-card}
What happens to draft pull requests in terms of merging ?
:::

:::{grid-item-card}
They cannot be merged.
:::

:::{grid-item-card}
Draft pull requests cannot be merged until they are converted to regular pull requests.
:::
