## Primary purpose

::::{grid}
:::{grid-item-card}
What is the primary purpose of GitHub Marketplace ?
:::

:::{grid-item-card}
To list free and paid tools for developers.
:::

:::{grid-item-card}
GitHub Marketplace serves as a platform where developers can discover, share, and acquire various tools and apps to enhance their GitHub workflows.\
It offers a wide range of both free and paid tools, including GitHub Actions and Apps, allowing developers to customize and improve their development processes.\
Whether you're looking for automation tools, integrations, or other resources, GitHub Marketplace provides a centralized hub for accessing these solutions.
:::
